Maven 1. build lifecycle定义 is the process for building and distributing a particular artifact (project) is clearly defined.
2. phase定义 is a step in the build lifecycle, which is an ordered sequence of phases. When a phase is given, Maven will execute every phase in the sequence up to and including the one defined.
3. plugin定义 is a collection of goals with a general common purpose.
4. goal定义 is similar to a Ant task.A goal can bound to zero or more build phases.
5.packaging定义 Some of the valid packaging values are jar, war, ear and pom,default is jar. different packaging type contains a different list of goals to bind to a particular phase.
