Java中的进制表示、进制转换、逻辑运算以及位运算符

Farubaba
2018-10-27
进制转换, 逻辑运算

一、进制语法

所有进制中，前缀中的0都是数字“0”，而不是字母“O”。

名称 前缀 示例 加法示例 描述
二进制 0b 0b10 = 2 0b10 + 010 = 10
0b10 + 10 = 12
0b10 + 0xa5 = 167
八进制 0 010 = 8 010 + 10 = 18
十进制 无 10 = 10 10 + 10 = 20
十六进制 0x 0xa5 = 165 0xa5 + 10 = 175

二、位运算

名称 运算符 运算规则 示例 描述
短路与 && 有0出0，全1出1 1010 && 1001 = 1000
非短路与 & 有0出0，全1出1 1010 & 1001 = 1000
短路或 \ \ 有1出1，全0出0 1010 \ \ 1001 = 1011
非短路或 \ 有1出1，全0出0 1010 \ 1001 = 1011
非 ~ 有1出0，有0出1 ~1010 = 0101
异或 ^ 相同得0，相异得1 1010 ^ 0110 = 1100

三、移位运算符

名称 运算符 运算规则 示例 描述
左移 << 左移运算是将一个二进制位的操作数按指定移动的位数向左移位，移出位被丢弃，右边的空位一律补0

num << 1,相当于num乘以2		 正数：0000000100010001 << 3 = 0000100010001000
负数（-4）： 1111111111111100 << 3 = 1111111111100000
右移 >> 右移运算是将一个二进制位的操作数按指定移动的位数向右移动，移出位被丢弃，左边移出的空位或者一律补0，或者补符号位，这由不同的机器而定.

通常：右移对于无符号类型强制补0，对于有符号类型续补符号位

num >> 1,相当于num除以2		 正数：0000000100010001 >> 4 = 0000000000010001
负数：1111111111111100 >> 4 = 1111111111111111
无符号右移 >>> 忽略符号位，空位都以0补齐 正数：0000000100010001 >>> 4 = 0000000000010001
负数：1111111111111100 >>> 4 = 0000111111111111

四、逻辑运算符

名称 运算符 运算规则 示例 描述
逻辑与 && 有false得false，全true得true true && true = true
true && false = false
false && false = false
逻辑或 \ \ 有true得true，全false得false true \ \ true = true
false \		 \ true = true
false \		 \ false = false
逻辑非 ! true得false，false得true !true = false
!false = true
逻辑异或 ^ 相同得false，不同得true true ^ true = false
true ^ false = true
