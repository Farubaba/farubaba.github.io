一、概念

  1. 过程式编程
  2. 函数式编程
  3. 面向对象编程

  4. 并发处理

    JavaScript 程序可以采用事件驱动（event-driven）和非阻塞式（non-blocking）设计，
在服务器端适合高并发环境，普通的硬件就可以承受很大的访问量。
